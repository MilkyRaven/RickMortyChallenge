## Features

- Episodes list with:
  - Infinite scrolling

  - Characters list with images
- Context + custom hooks for state management
- Basic error handling and loading states



## Getting Started

```bash
# install dependencies
bun install

# run on iOS
bun run ios

# run on Android
bun run android
```
